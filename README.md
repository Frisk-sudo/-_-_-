## Описание проекта

Симулятор позволяет создавать организмы (например, зайцев и лис), добавлять их в экосистему и запускать пошаговую симуляцию. На каждом шаге (дне) организмы потребляют ресурсы, стареют и могут погибать. Модель демонстрирует базовые принципы взаимодействия видов в замкнутой среде.

**Ключевые особенности:**
* Объектно‑ориентированная архитектура с разделением на модули.
* Поддержка разных типов организмов с уникальным поведением.
* Пошаговая симуляция с отслеживанием состояния каждого организма.
* Соответствие стандартам PEP8.
* Наличие документации и комментариев к коду.

## Цели и задачи

**Цели:**
* Научиться проектировать простую систему моделирования экосистемы.
* Освоить базовые принципы ООП.
* Развить навыки работы с Git при командной разработке.
* Закрепить понимание стандартов оформления кода (PEP8).
* Научиться писать документацию и комментировать код.

**Задачи:**
* Создать репозиторий на GitHub/GitLab для совместной работы.
* Реализовать минимум 3 класса: `Organism`, `Population`, `Ecosystem`.
* Разделить проект на логические модули.
* Написать документацию и комментарии к каждому классу и функции.
* Протестировать базовую логику взаимодействия между объектами.
* Выполнить pull request с code review.
